# Glossário — Módulo 4

| Termo | Definição |
|-------|-----------|
| **On-Demand** | Pagamento por uso sem compromisso. |
| **Reserved Instance (RI)** | Compromisso 1–3 anos para desconto. |
| **Savings Plan** | Compromisso flexível de uso ($/hora). |
| **Spot** | Instância com até 90% off, interruptível. |
| **Dedicated Host** | Servidor físico dedicado (BYOL). |
| **Free Tier** | Uso gratuito AWS (12 meses, sempre grátis, trials). |
| **TCO** | Total Cost of Ownership. |
| **Pricing Calculator** | Estimador oficial de custos AWS. |
| **Cost Explorer** | Análise visual de custos. |
| **Budgets** | Orçamentos e alertas. |
| **CUR** | Cost and Usage Report — relatório detalhado. |
| **Cost Allocation Tag** | Tag que aparece na fatura. |
| **Consolidated Billing** | Fatura única para várias contas. |
| **Management Account** | Conta pagadora no Organizations. |
| **TAM** | Technical Account Manager. |
| **Trusted Advisor** | Verificador de boas práticas (5 categorias). |
| **re:Post** | Comunidade oficial AWS. |
| **APN** | AWS Partner Network. |
| **AMS** | AWS Managed Services. |
| **IEM** | Infrastructure Event Management. |
