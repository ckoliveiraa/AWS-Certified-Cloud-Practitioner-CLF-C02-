# 4.2 — Ferramentas de Faturamento e Custos

## AWS Billing & Cost Management

- Console de **faturas, pagamentos e preferências fiscais**.
- Histórico de cobrança mês a mês.

## AWS Cost Explorer

- **Visualização gráfica** de custos e uso.
- Filtros por serviço, tag, conta, região.
- Previsões de custo até **12 meses** à frente.
- Gera **recomendações de Savings Plans / Reserved Instances** com base no histórico.

## CloudWatch Billing Alarms

- Alarmes baseados na métrica **EstimatedCharges** do CloudWatch.
- ⚠ A métrica só é publicada na região **us-east-1** — o alarme **deve ser criado em us-east-1**.
- Anterior ao Budgets; hoje **Budgets é o recomendado** (mais flexível), mas Billing Alarms ainda cai no exame.

## AWS Budgets

- Define **orçamentos** e **alertas** (email, SNS).
- Tipos: custo, uso, reservas, Savings Plans.
- **2 orçamentos grátis**; US$ 0,02/dia/orçamento extra.

## AWS Cost & Usage Report (CUR)

- **Relatório mais detalhado** (nível de hora e recurso).
- Entregue em **S3**; analisado com **Athena, Redshift, QuickSight**.

## AWS Cost Allocation Tags

- **Tags** que aparecem no relatório de cobrança.
- Tipos: **AWS-generated** e **User-defined**.
- Essenciais para rastrear custos por projeto/equipe.

## AWS Cost Categories

- Agrupa custos por **regras personalizadas** (ex.: por time, ambiente, produto).
- Aparece no Cost Explorer, Budgets e CUR como uma dimensão a mais.

## AWS Billing Conductor

- Permite **customizar a fatura** mostrada a unidades de negócio ou clientes finais (resellers).
- Cria *billing groups* com pricing customizado — não altera o que a AWS cobra do payer, só como é repartido.

## AWS Cost Anomaly Detection

- **ML** detecta gastos anormais.
- Alertas via SNS ou email.

## AWS Compute Optimizer

- Recomenda **rightsizing** (EC2, EBS, Lambda, ASG).
- Identifica recursos subutilizados.

## AWS Trusted Advisor — pilar Cost Optimization

- Checks específicos de custo: **EC2 ociosas**, **RIs subutilizadas**, **Elastic IPs não associados**, **load balancers ociosos**, **volumes EBS sem uso**.
- Visão geral de oportunidades de economia (visão complementar ao Compute Optimizer).

## Free Tier Usage Alerts

- Configurados no **Billing Console** → preferências.
- Notificam por email quando o uso se aproxima do limite gratuito de algum serviço.

## AWS Pricing Calculator *(revisão)*
- Estimativa de custo antes de provisionar.

## AWS Marketplace

- Catálogo de software de terceiros — cobrança integrada à fatura AWS.

---

## Resumo "qual usar"?

| Pergunta | Ferramenta |
|----------|-----------|
| Quanto gastei por serviço? | **Cost Explorer** |
| Alerta se passar de R$X | **Budgets** |
| Relatório detalhado hora a hora | **CUR** |
| Gasto suspeito? | **Cost Anomaly Detection** |
| Preciso rightsizing | **Compute Optimizer** |
| Estimar antes de criar | **Pricing Calculator** |
| Separar custo por time | **Cost Allocation Tags** / **Cost Categories** |
| Alarme de billing via CloudWatch | **CloudWatch Billing Alarm** (em us-east-1) |
| Encontrar EC2 ociosa, EIP solto, RI subutilizada | **Trusted Advisor** (Cost Optimization) |
| Faturar clientes/BUs com pricing próprio | **Billing Conductor** |

## Pontos-Chave para o Exame

- ✅ **Cost Explorer** = análise visual (dashboards).
- ✅ **Budgets** = alertas proativos.
- ✅ **CUR** = granularidade máxima (hora/recurso).
- ✅ **Tags** são essenciais para rastrear custos.
- ✅ **Compute Optimizer** → rightsizing.
- ✅ **Trusted Advisor** tem pilar de **Cost Optimization** (EC2 ociosa, EIP solto, RI subutilizada).
- ✅ **Billing Alarm via CloudWatch** só funciona em **us-east-1**.
- ✅ **Billing Conductor** = customizar fatura para BUs/clientes (resellers).
- ✅ **Cost Categories** = agrupar custos por regras.
