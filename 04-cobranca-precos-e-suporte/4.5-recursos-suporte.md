# 4.5 — Recursos de Suporte e Comunidade

## AWS Support Center

- Portal para abrir **tickets** (cases).
- Disponível a partir do plano **Developer**.

## AWS Trusted Advisor

**6 categorias** de checks (alinhadas ao Well-Architected):
1. Otimização de custos
2. Performance
3. Segurança
4. Tolerância a falhas
5. Limites de serviço
6. Operational Excellence

*(Checks completos a partir do plano Business; Basic/Developer têm apenas 7 core checks.)*

> ⚠ **6 categorias ≠ 7 core checks.** As **6 categorias** são os temas em que os checks se organizam. Os **7 core checks** são checks **individuais** específicos liberados para Basic/Developer — todos dentro de apenas 2 categorias (Segurança e Limites de Serviço). Business+ libera **todos** os checks das 6 categorias.

### Os 7 core checks (Basic/Developer)

| # | Check | Categoria |
|---|-------|-----------|
| 1 | Service Limits | Limites de serviço |
| 2 | Security Groups – Specific Ports Unrestricted | Segurança |
| 3 | IAM Use | Segurança |
| 4 | MFA on Root Account | Segurança |
| 5 | EBS Public Snapshots | Segurança |
| 6 | RDS Public Snapshots | Segurança |
| 7 | S3 Bucket Permissions | Segurança |

## AWS Well-Architected Tool

- Ferramenta **gratuita** no console.
- Permite fazer **reviews** da sua workload contra os **6 pilares** do Well-Architected Framework.
- Gera plano de melhoria com riscos identificados (HRI/MRI).

## AWS Health Dashboard

Painel unificado com duas visões:
- **Service health** — status **público** dos serviços AWS (era *Service Health Dashboard*).
- **Your account health** — eventos que afetam **sua conta** (era *Personal Health Dashboard*).

## AWS re:Post

- Comunidade oficial tipo **Stack Overflow**, moderada pela AWS.
- **Gratuito**.

## AWS Knowledge Center

- **Base de artigos** oficiais para problemas comuns.

## AWS Training & Certification

| Recurso | Público |
|---------|---------|
| **AWS Skill Builder** | Indivíduos — cursos gratuitos + assinaturas pagas (labs, jogos). |
| **AWS Educate** | Estudantes 13+ — autoatendimento, sem cartão. |
| **AWS Academy** | **Instituições de ensino** — currículo formal para professores. |

## AWS Whitepapers, Architecture Center e Guides

- **AWS Whitepapers & Guides** — documentos técnicos (inclui o **Well-Architected Framework whitepaper**).
- **AWS Architecture Center** — arquiteturas de referência por caso de uso/indústria.
- **AWS Prescriptive Guidance** — guias passo-a-passo oficiais para cenários comuns (migração, modernização).
- **AWS Solutions Library** — soluções prontas para implantar (reference deployments).

## AWS Professional Services

- Consultoria **paga** da AWS para projetos complexos.

## AWS Partner Network (APN)

- **Consulting Partners** — implementam soluções.
- **Technology Partners** — ISVs/SaaS.
- Tiers de parceiros: **Select → Advanced → Premier** (conforme expertise/volume).

## AWS IQ

- Contrate **freelancers certificados** em AWS.
- Para projetos curtos.

## AWS Managed Services (AMS)

- AWS gerencia sua infraestrutura (operação, patches, segurança).

## AWS Abuse Team

- Receba / reporte **abuse@amazonaws.com** — atividade maliciosa.

## Resumo "onde pedir ajuda?"

| Situação | Recurso |
|----------|---------|
| Dúvida técnica geral | **re:Post / Knowledge Center** |
| Otimização de conta | **Trusted Advisor** |
| Problema na conta | **Support Center** (ticket) |
| Contratar freelancer | **AWS IQ** |
| Projeto grande | **Professional Services / APN** |
| Delegar operação inteira | **AMS** |
| Treinamento | **Skill Builder** |
| Avaliar workload contra Well-Architected | **AWS Well-Architected Tool** |
| Arquitetura de referência por caso de uso | **AWS Architecture Center** |
| Solução pronta para implantar | **AWS Solutions Library** |
| Guia passo-a-passo oficial | **AWS Prescriptive Guidance** |
| Atividade maliciosa | **AWS Abuse Team** |

## Pontos-Chave para o Exame

- ✅ **re:Post** = comunidade gratuita.
- ✅ **Trusted Advisor** = **6 categorias** (incluindo Operational Excellence).
- ✅ **AWS IQ** = freelancers; **APN** = parceiros; **Professional Services** = consultoria AWS.
- ✅ **AMS** = AWS opera sua infra.
- ✅ **AWS Health Dashboard** unifica Service health (público) e **Your account health** (sua conta — antigo Personal Health Dashboard).
- ✅ **Well-Architected Tool** = review gratuito da workload contra os 6 pilares.
- ✅ **Skill Builder** (indivíduos) ≠ **Educate** (estudantes) ≠ **Academy** (instituições).
