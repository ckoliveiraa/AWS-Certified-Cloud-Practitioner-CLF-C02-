# Módulo 4 — Cobrança, Preços e Suporte

> **Domínio 4 do exame CLF-C02** — Peso: **12%**

Este módulo cobre modelos de preço, ferramentas de faturamento, planos de suporte e recursos de relacionamento com a AWS.

---

## Objetivos de Aprendizagem

- Entender os **modelos de preço** da AWS (On-Demand, Reserved, Spot, Savings Plans).
- Usar **AWS Pricing Calculator** e estimar **TCO**.
- Utilizar **Billing & Cost Management**, **Cost Explorer**, **Budgets**, **CUR**.
- Aplicar **Consolidated Billing** via Organizations.
- Diferenciar os **5 planos de suporte** AWS.
- Conhecer recursos como **Trusted Advisor, Support Center, TAM, AWS IQ, re:Post, Knowledge Center**.
