# Glossário — Módulo 1

| Termo | Definição |
|-------|-----------|
| **CapEx** | *Capital Expenditure* — despesa de capital (compra de ativos). |
| **OpEx** | *Operational Expenditure* — despesa operacional (pagamento por uso). |
| **Elasticidade** | Capacidade de aumentar **e diminuir** recursos automaticamente conforme a demanda. |
| **Escalabilidade** | Capacidade de crescer para atender demanda (vertical ou horizontalmente). |
| **IaaS** | *Infrastructure as a Service* — infraestrutura gerenciada (ex.: EC2). |
| **PaaS** | *Platform as a Service* — plataforma gerenciada (ex.: Elastic Beanstalk). |
| **SaaS** | *Software as a Service* — software pronto (ex.: Amazon WorkMail). |
| **Nuvem Pública** | Recursos compartilhados via internet (AWS, Azure, GCP). |
| **Nuvem Privada** | Infra dedicada a uma única organização. |
| **Nuvem Híbrida** | Combinação de on-premises + nuvem pública. |
| **Pay-as-you-go** | Modelo de cobrança por consumo. |
| **Well-Architected Framework** | Conjunto de 6 pilares para arquiteturas AWS. |
| **6 Rs** | Estratégias de migração: Rehost, Replatform, Repurchase, Refactor, Retire, Retain. |
| **CAF** | *Cloud Adoption Framework* — guia de adoção da nuvem em 6 perspectivas. |
| **TCO** | *Total Cost of Ownership* — custo total de propriedade. |
