# 1.1 — O que é Computação em Nuvem

## Definição Oficial da AWS

> *"Computação em nuvem é a entrega sob demanda de poder computacional, banco de dados, armazenamento, aplicações e outros recursos de TI via internet, com precificação pay-as-you-go."*

---

## Características Essenciais

| Característica | Descrição |
|---------------|-----------|
| **Sob demanda** | Provisione recursos quando precisar, sem intervenção humana. |
| **Autoatendimento** | Console, CLI ou API — você escolhe. |
| **Elástico** | Escala automaticamente com a demanda. |
| **Medido** | Pague apenas pelo que usar. |
| **Amplo acesso** | Disponível via internet, em qualquer lugar. |

---

## Antes vs. Depois da Nuvem

| Tradicional (On-Premises) | Nuvem AWS |
|--------------------------|-----------|
| Compra de servidores físicos | Provisionamento em minutos |
| Superprovisionamento "por garantia" | Elasticidade sob demanda |
| Equipe dedicada a data center | Foco no negócio |
| CapEx alto inicial | OpEx variável |
| Escala global = meses | Escala global = cliques |

---

## Pontos-Chave para o Exame

- ✅ Nuvem = **sob demanda + pay-as-you-go + via internet**.
- ✅ Autoatendimento é um pilar — sem abrir ticket.
- ✅ Medição de uso é o que habilita o modelo de cobrança.
