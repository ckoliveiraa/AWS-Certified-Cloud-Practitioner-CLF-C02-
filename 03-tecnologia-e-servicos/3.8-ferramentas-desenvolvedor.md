# 3.8 — Ferramentas para Desenvolvedores

## Suíte "Code"

| Serviço | Função |
|---------|--------|
| **CodeCommit** | Repositório Git privado (descontinuado para novos clientes — use GitHub/GitLab) |
| **CodeBuild** | Build e testes |
| **CodeDeploy** | Deploy em EC2, ECS, Lambda, on-premises |
| **CodePipeline** | Orquestração CI/CD |
| **CodeArtifact** | Repositório de pacotes (npm, Maven, PyPI) |
| **CodeGuru** | Revisão automatizada de código + profiler |
| **CodeStar** | Templates de projeto (descontinuado) |

---

## AWS Cloud9

- **IDE baseada no navegador** (descontinuada para novos clientes).
- Colaborativa, integrada com Lambda e terminal.

## AWS X-Ray

- **Tracing distribuído** — rastreia requests em microsserviços.
- Identifica gargalos e erros.

## AWS AppConfig

- Gerencia **feature flags** e configurações em tempo real.

## AWS SDK e CLI

- **SDKs** para Python (boto3), JS, Java, Go, .NET, Ruby, PHP.
- **AWS CLI** — terminal.
- **AWS CloudShell** — shell no navegador com CLI pré-instalada.

---

## Pontos-Chave para o Exame

- ✅ **CodePipeline** orquestra; **CodeBuild** builda; **CodeDeploy** implanta.
- ✅ **X-Ray** = tracing para debug distribuído.
- ✅ **CloudShell** = terminal grátis no navegador.
- ✅ **SDK** para seu código; **CLI** para scripts.
