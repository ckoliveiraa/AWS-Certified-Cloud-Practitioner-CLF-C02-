# Glossário — Módulo 3

| Termo | Definição |
|-------|-----------|
| **Região** | Localização geográfica com múltiplas AZs. |
| **AZ** | Availability Zone — data center isolado dentro de uma região. |
| **Edge Location** | PoP da AWS para CloudFront e Route 53. |
| **AMI** | Amazon Machine Image — template de instância EC2. |
| **Spot** | Instância EC2 com até 90% de desconto; pode ser interrompida. |
| **Reserved Instance (RI)** | Compromisso de 1–3 anos com desconto. |
| **Savings Plan** | Compromisso flexível de uso. |
| **Fargate** | Compute serverless para containers. |
| **Bucket** | Container de objetos no S3. |
| **Glacier** | Classes S3 para arquivamento. |
| **Snapshot** | Backup pontual (EBS). |
| **Multi-AZ** | Redundância entre AZs (HA). |
| **Read Replica** | Réplica de leitura (RDS/Aurora). |
| **VPC** | Virtual Private Cloud — rede isolada. |
| **IGW** | Internet Gateway. |
| **NAT** | Network Address Translation. |
| **CDN** | Content Delivery Network. |
| **ELB** | Elastic Load Balancer. |
| **ALB / NLB / GLB** | Application / Network / Gateway Load Balancer. |
| **IaC** | Infrastructure as Code. |
| **Stack** | Grupo de recursos gerenciados por CloudFormation. |
| **Pub/Sub** | Publish-Subscribe (SNS). |
| **ETL** | Extract-Transform-Load. |
