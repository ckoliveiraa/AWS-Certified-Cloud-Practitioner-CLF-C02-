# 3.6 — Monitoramento e Gestão

## Amazon CloudWatch

- Métricas, logs, eventos, alarmes, dashboards.
- **Métricas customizadas** via API.
- **Logs Insights** para consultas.

## AWS CloudTrail *(visto no Domínio 2)*
- Log de chamadas de API.

## AWS Config *(visto no Domínio 2)*
- Histórico de configurações.

## AWS Trusted Advisor

5 categorias de checks:
1. **Otimização de Custos**
2. **Performance**
3. **Segurança**
4. **Tolerância a Falhas**
5. **Limites de Serviço**

Mais checks disponíveis nos planos **Business** e **Enterprise**.

## AWS Health Dashboard

- **Service Health Dashboard** — status público dos serviços.
- **Personal Health Dashboard** — eventos que afetam **sua conta**.

---

## AWS CloudFormation

- **Infrastructure as Code (IaC)** — JSON/YAML.
- **Stacks** e **Change Sets**.
- Rollback automático em falhas.
- **Gratuito** (paga apenas pelos recursos).

## AWS CDK (Cloud Development Kit)

- IaC usando **linguagens de programação** (TypeScript, Python, Java…).
- Gera templates CloudFormation.

## AWS Systems Manager (SSM)

Suíte de operações:
- **Session Manager** — acesso shell sem SSH/RDP.
- **Patch Manager** — patches em massa.
- **Parameter Store** — configs e segredos (simples).
- **Run Command** — executa comandos em frota.
- **Automation** — runbooks.

## AWS OpsWorks

- Gerencia Chef/Puppet na AWS.

## AWS Service Catalog

- Catálogo de produtos aprovados para self-service em empresas.

## AWS License Manager

- Gerencia licenças de software (Oracle, SQL Server, SAP, etc.).

---

## Pontos-Chave para o Exame

- ✅ **CloudFormation = IaC em JSON/YAML**; **CDK = IaC em código**.
- ✅ **Trusted Advisor** tem **5 categorias**.
- ✅ **Session Manager** elimina necessidade de SSH.
- ✅ **Personal Health Dashboard** é específico da sua conta.
