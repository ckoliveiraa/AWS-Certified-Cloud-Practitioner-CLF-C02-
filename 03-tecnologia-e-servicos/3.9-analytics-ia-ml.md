# 3.9 — Analytics, IA e Machine Learning

## Analytics

| Serviço | Função |
|---------|--------|
| **Amazon Athena** | Queries SQL sobre dados no **S3** (serverless) |
| **AWS Glue** | **ETL serverless** + catálogo de dados |
| **Amazon EMR** | Hadoop, Spark, Presto gerenciados |
| **Amazon Kinesis** | Streaming de dados em tempo real |
| **Amazon MSK** | Kafka gerenciado |
| **Amazon QuickSight** | **BI / dashboards** |
| **AWS Data Exchange** | Marketplace de datasets |
| **Amazon OpenSearch** | Busca e analytics (fork Elasticsearch) |

### Kinesis — 4 sabores
- **Data Streams** — ingestão real-time
- **Data Firehose** — entrega para S3, Redshift, OpenSearch
- **Data Analytics** — SQL sobre streams
- **Video Streams** — vídeo ao vivo

## IA / ML de Alto Nível (sem conhecer ML)

| Serviço | Função |
|---------|--------|
| **Amazon Rekognition** | Visão computacional (imagens/vídeos) |
| **Amazon Polly** | Texto → fala |
| **Amazon Transcribe** | Fala → texto |
| **Amazon Translate** | Tradução de idiomas |
| **Amazon Comprehend** | NLP (sentimento, entidades) |
| **Amazon Lex** | Chatbots (motor do Alexa) |
| **Amazon Textract** | Extrai texto de documentos/PDFs |
| **Amazon Personalize** | Recomendações |
| **Amazon Forecast** | Previsão de séries temporais |
| **Amazon Kendra** | Busca empresarial inteligente |
| **Amazon Fraud Detector** | Detecção de fraude |

## ML Plataforma

- **Amazon SageMaker** — plataforma **end-to-end** para ML (build, train, deploy).

## IA Generativa

- **Amazon Bedrock** — modelos de fundação (Claude, Llama, Titan, etc.) via API gerenciada.
- **Amazon Q** — assistente de IA generativa para negócios e devs.

---

## Pontos-Chave para o Exame

- ✅ **Athena = SQL no S3** serverless.
- ✅ **Glue = ETL** serverless.
- ✅ **QuickSight = BI**.
- ✅ **Rekognition** (imagem), **Polly** (TTS), **Transcribe** (STT), **Translate**, **Comprehend** (NLP).
- ✅ **SageMaker** = plataforma ML completa.
- ✅ **Bedrock** = IA generativa (modelos de fundação).
