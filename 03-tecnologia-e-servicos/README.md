# Módulo 3 — Tecnologia e Serviços de Nuvem

> **Domínio 3 do exame CLF-C02** — Peso: **34%** (o maior do exame!)

Este módulo cobre a infraestrutura global da AWS e os principais serviços de computação, armazenamento, banco de dados, rede, monitoramento, desenvolvimento e IA/ML.

---

## Objetivos de Aprendizagem

- Entender a **infraestrutura global** da AWS (Regiões, AZs, Edge).
- Identificar serviços de **computação, armazenamento, bancos de dados e rede**.
- Conhecer serviços de **monitoramento, automação e desenvolvimento**.
- Conhecer serviços de **analytics, IA/ML e integração**.
