# 2.5 — Proteção de Rede e Aplicações

## AWS WAF — Web Application Firewall

- Protege aplicações web contra **exploits comuns** (SQL injection, XSS).
- Funciona em: **CloudFront, ALB, API Gateway, AppSync**.
- Regras baseadas em: IP, país, headers, body, rate limiting.

---

## AWS Shield

Proteção contra **DDoS**.

| Plano | Preço | Recursos |
|-------|-------|----------|
| **Shield Standard** | **Grátis** (ativado automaticamente) | Proteção L3/L4 básica |
| **Shield Advanced** | USD 3.000/mês | L7, DRT, reembolso de custos DDoS, WAF incluso |

---

## AWS Firewall Manager

- Gerencia **WAF, Shield e SGs** centralmente em várias contas via **Organizations**.

---

## AWS Network Firewall

- Firewall **stateful** para **VPCs** (nível de rede).
- Inspeção profunda, regras Suricata.

---

## Security Groups vs NACLs *(revisão rápida)*

| Recurso | Security Group | NACL |
|---------|----------------|------|
| Nível | Instância | Subnet |
| Stateful? | ✅ Sim | ❌ Não |
| Regras | Allow apenas | Allow + Deny |
| Avaliação | Todas as regras | Ordem numérica |

---

## Pontos-Chave para o Exame

- ✅ **WAF** = camada 7 (aplicação); **Shield** = DDoS.
- ✅ **Shield Standard é grátis e automático**.
- ✅ **Shield Advanced** inclui WAF.
- ✅ **Firewall Manager** = gestão centralizada multi-conta.
- ✅ **SG** = stateful; **NACL** = stateless.
