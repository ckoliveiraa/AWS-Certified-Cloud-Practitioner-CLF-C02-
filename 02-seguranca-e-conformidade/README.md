# Módulo 2 — Segurança e Conformidade

> **Domínio 2 do exame CLF-C02** — Peso: **30%**

Este módulo cobre o modelo de responsabilidade compartilhada, IAM, serviços de segurança, criptografia, conformidade e auditoria.

---

## Objetivos de Aprendizagem

- Aplicar o **Modelo de Responsabilidade Compartilhada**.
- Configurar **IAM** (usuários, grupos, roles, policies, MFA).
- Usar **AWS Organizations**, SCPs e Control Tower.
- Identificar serviços de **criptografia e proteção**: KMS, CloudHSM, Secrets Manager, ACM.
- Entender serviços de **detecção e resposta**: GuardDuty, Inspector, Macie, Security Hub, Detective.
- Utilizar **CloudTrail, CloudWatch, Config** para auditoria e conformidade.
- Conhecer **AWS Artifact** e relatórios de conformidade.
