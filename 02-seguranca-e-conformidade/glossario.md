# Glossário — Módulo 2

| Termo | Definição |
|-------|-----------|
| **IAM** | Identity and Access Management — gerencia identidades e permissões. |
| **MFA** | Multi-Factor Authentication — senha + segundo fator. |
| **Policy** | Documento JSON que define permissões. |
| **Role** | Identidade temporária assumida por usuários/serviços. |
| **SCP** | Service Control Policy — guardrail em Organizations. |
| **OU** | Organizational Unit — agrupamento de contas. |
| **KMS** | Key Management Service. |
| **CMK / KMS Key** | Chave gerenciada pelo KMS. |
| **HSM** | Hardware Security Module. |
| **ACM** | AWS Certificate Manager — certificados SSL/TLS. |
| **WAF** | Web Application Firewall — camada 7. |
| **DDoS** | Distributed Denial of Service. |
| **SG** | Security Group — firewall stateful de instância. |
| **NACL** | Network ACL — firewall stateless de subnet. |
| **GuardDuty** | Detecção de ameaças baseada em ML. |
| **Inspector** | Escaneamento de vulnerabilidades. |
| **Macie** | Descoberta de PII no S3. |
| **Security Hub** | Dashboard central de segurança. |
| **CloudTrail** | Log de chamadas de API. |
| **Config** | Histórico de configurações. |
| **Artifact** | Portal de relatórios de conformidade. |
